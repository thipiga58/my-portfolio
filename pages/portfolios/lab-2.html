Introduction

This lab demonstrates the creation of a virtual event management system using XML and related technologies. It includes the structure of events, organizations, hosts, participants, and resources, along with DTD validation.

DTD Code

            <!ELEMENT events (organization, event+)>
            <!ELEMENT organization (orgName, logo, website?, socialMediaLink*)>
            <!ELEMENT orgName (#PCDATA)>
            <!ELEMENT logo (#PCDATA)>
            <!ELEMENT website (#PCDATA)>
            <!ELEMENT socialMediaLink (#PCDATA)>
            <!ELEMENT event (eventName, date, hosts, participants, resources, feedback*)>
            <!ATTLIST event eventID ID #REQUIRED>
            <!ELEMENT eventName (#PCDATA)>
            <!ELEMENT date EMPTY>
            <!ATTLIST date start CDATA #REQUIRED end CDATA #REQUIRED>
            <!ELEMENT hosts (host+)>
            <!ELEMENT host (hostName, contactInfo)>
            <!ATTLIST host hostID ID #REQUIRED>
            <!ELEMENT hostName (#PCDATA)>
            <!ELEMENT contactInfo (phone, email)>
            <!ELEMENT phone (#PCDATA)>
            <!ELEMENT email (#PCDATA)>
            <!ELEMENT participants (participant+)>
            <!ELEMENT participant (participantName, role, contact)>
            <!ELEMENT participantName (#PCDATA)>
            <!ELEMENT role (#PCDATA)>
            <!ELEMENT contact (phone, email)>
            <!ELEMENT resources (venue, equipment+, catering?)>
            <!ELEMENT venue (location, capacity)>
            <!ELEMENT location (#PCDATA)>
            <!ELEMENT capacity (#PCDATA)>
            <!ELEMENT equipment (#PCDATA)>
            <!ELEMENT catering (#PCDATA)>
            <!ELEMENT feedback (comment)>
            <!ATTLIST feedback feedbackID ID #REQUIRED>
            <!ELEMENT comment (#PCDATA)>
