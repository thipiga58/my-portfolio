My First Flutter App

In this project, I built my first Flutter app that displays personalized details such as my name, banner ID, class and section, and my favorite quote. The app was developed using Flutter and tested on both the default Android emulator and an external emulator.

Key Features:

  • Displays personalized information (Name, Banner ID, Class & Section, Favorite Quote)
  • Customization of layout, colors, and fonts
  • Emulator testing on both default and external Android emulators
