Human Body Trivia

This app, built using MIT App Inventor, features two screens: a welcome page and a quiz page. The quiz includes three questions with interactive input types like checkboxes and dropdown menus. Users can navigate between screens and see their final score calculated dynamically upon submission.

Key Features:

  • Two screens: Welcome page and quiz page
  • Interactive input types: Checkboxes and dropdown menus
  • Dynamic score calculation
  • User-friendly and engaging design
