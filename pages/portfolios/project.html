Project

Nostalgia Quest

Nostalgia Quest is a multi-game mobile app that revives classic games—Chess, Tetris, and Snake—offering users a blend of strategy, puzzle-solving, and arcade fun. The app is designed with both light and dark modes for optimal user experience and built entirely using Android Studio and Flutter, ensuring a modern and responsive interface.

Key Features:

  • Three iconic games: Chess, Tetris, and Snake
  • Switchable light and dark mode for user convenience
  • Informative content on game rules, scoring, and historical origins
  • Intuitive navigation with a dedicated information page accessible from the home screen
  • Developed using Android Studio and Flutter for seamless cross-platform performance

Reflection:

Creating Nostalgia Quest was a rewarding experience, allowing me to combine technical development with user-centric design. The app was built entirely using Android Studio and Flutter, which provided a robust framework for implementing multi-game functionality and customizable themes. I learned to integrate multiple games into a single app, ensuring smooth transitions and consistent performance across game modes. Designing the light and dark mode functionality challenged me to think about accessibility and aesthetic preferences. Incorporating the historical and scoring information for each game added an educational element that I thoroughly enjoyed crafting. This project strengthened my skills in app architecture, UI/UX design, and interactive programming.
